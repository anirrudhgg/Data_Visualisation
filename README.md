# Data_Visualisation
This repository contains key projects completed during my Data Visualization course. These projects demonstrate my ability to process complex datasets and create insightful visual representations using Python. The Datasets are processed into visual information from raw data making them easier to comprehend. The work spans biological data analysis, financial trend tracking, and a final data report on the topic of "High Technology Exports 2010-24 (in $Billion).


1. Project 1: Gene Expression Analysis

Core Task: Analyzing cell specification patterns using gene expression datasets.

Key Visuals: Mapping gene behavior and exploring Waddington’s epigenetic landscape.


2. Project 2: Stock Market Visualization

Core Task: Visualizing stock market volatility and performance trends.

Tech: Data visualization libraries (Matplotlib/Seaborn) for time-series analysis.


3. Final Project: Interactive Data Report

Link: https://lookerstudio.google.com/s/swHT_q2vhso  (Click this link to view the Data Report on google looker studio)

Description: A comprehensive data report created as the final project for my Data Visualisation Course . It utilizes Google Looker Studio to transform raw data into an interactive, stakeholder-ready dashboard. The topic is "High Technology exports 2010-24 (in $Billion)" and it was chosen from the world data bank. The report compares this metric against 7 major countries India, China, Russia, Japan, France, USA and Germany.
